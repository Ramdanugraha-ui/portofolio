# 🌐 Visual Portfolio & Digital CV - Ramdan Nugraha, S.Kom

[![Website Status](https://img.shields.io/website?url=https%3A%2F%2Fportofolio-two-sable-40.vercel.app%2F&version=2026)](https://portofolio-two-sable-40.vercel.app/)
[![Tech Stack](https://img.shields.io/badge/Tech_Stack-HTML5%20%7C%20TailwindCSS%20%7C%20JavaScript-blue)](#-tech-stack)

Website ini merupakan portofolio visual interaktif dan CV digital resmi saya sebagai lulusan Sistem Informasi. Didesain dengan konsep _modern dark mode dashboard_ untuk menyajikan rekam jejak akademik, keterampilan teknis, serta bukti dokumentasi proyek nyata secara transpiratif kepada rekruter.

🔗 **Live Preview:** [portofolio-two-sable-40.vercel.app](https://portofolio-two-sable-40.vercel.app/)

---

## 🚀 Fitur Utama Web

- **Hero Dashboard Stats:** Menampilkan metrik utama (IPK 3.77, Jumlah Proyek, & Sertifikasi) langsung saat halaman pertama dimuat.
- **Interactive Proficiency Bars:** Indikator tingkat kemahiran _hard skills_ dan bahasa dengan visual 5-garis _glow effect_ bergaya dasbor modern.
- **3D Coverflow Carousel (Swiper.js):** Slider dokumentasi visual interaktif 3D pada linimasa pengalaman untuk memamerkan bukti proyek kerja praktek dan organisasi tanpa merusak _layout_ utama.
- **Direct WA & Link Integration:** Integrasi tombol kontak langsung ke WhatsApp Chat room dan tautan validasi eksternal (seperti portal berita resmi kampus)[cite: 1].

## 🛠️ Tech Stack & Libraries

- **Core:** HTML5, Tailwind CSS v4 (via CDN), Vanilla JavaScript
- **Icons:** FontAwesome v6
- **Slider Engine:** Swiper.js v11 (Coverflow 3D Effect)
